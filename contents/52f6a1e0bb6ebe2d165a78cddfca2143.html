CSU02 - Ordem de compra

UMLUseCase
Untitled :: Model1 :: CSU02 - Ordem de compra

Description

  1. Usuário faz análise das ações que deseja comprar
  2. Usuário define quantidade da compra (fracionário ou em lote)
  3. Usuário define o tipo de ordem (com ou sem gatilho)
  4. Usuário envia a OC para corretora
  5. Corretora permite o envio de OC
  6. Corretora envia a OC para B3
  7. Corretora retorna a execução da ordem
  8. Usuário visualiza o status da OC
  9. Usuário se quiser pode modificar a OC

9.1 Usuário consulta o histórico de OC 9.2 Usuário cancela a OC 9.3 Corretora devolve o valor do ativo após análise para o usuário
